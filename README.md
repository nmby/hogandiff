# hogandiff

hogandiff（方眼Diff）はExcelファイル同士の差分検出ツールです。<br>
次の特徴があります。
* <b>無料</b>で利用できます。
* <b>挿入／削除された行や列を検出</b>することができます。
* <b>簡単</b>に使用できます。必要最小限の機能に絞っていますので、パソコン初心者の方でも直感的に利用することができます。

詳しくは次のサイトをご覧ください。<br>
<http://hogandiff.hotchpotch.xyz/>
